== Description ==

http://www.atbar.se

ATbar är ett verktygsfält för ökad tillgänglighet som kan adderas till en användares webbläsare, alernativt kan en startknapp för ATbar läggas in på webbplatsen. ATbar har stöd för svenska, engelska och arabiska.

ATbar erbjuder en uppsättning användarinställningar för att interagera med webbsidor, till exempeel större text, större radavstånd. ändrad font-stil, ändrad bakgrundsfärg, ändrad textfärg och text till tal.

Om du behöver andra funktioner än de som redan finns på ATbar, kontakta erik@ictenablers.com

== Installation ==

1. Ladda upp svensk-atbar.zip i ditt administrationsgränsnitt under menyn "Tillägg".
2. Aktivera ATbar under menyn "Tillägg".
3. Gör inställningar under menyn "Inställlningar", välj Svensk ATbar.
